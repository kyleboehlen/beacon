# In-Person Play Design Notes

## Physical Components (app must NOT replace)
- **Physical dice** -- players roll real dice; app should not provide digital dice
- **Physical cards** -- players draw real cards; app should not replace card drawing
- **Physical alien chits** -- players draw real alien chits; app should not replace chit drawing

## Session Limits
- Only allow **one active session** for the shared map at any given point in time
- You should play together in person with a shared map
